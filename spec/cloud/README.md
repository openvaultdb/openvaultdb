## Guiding Principle

> The official OpenVaultDB Cloud service is an instance of the same Apache 2.0 open-source reference implementation available for self-hosting and third-party hosting.

Commercial value comes from managed hosting, operations, monitoring, support, and SLAs — not from proprietary server functionality or protocol extensions.
